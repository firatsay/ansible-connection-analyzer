# Security Policy

## Overview

This repository contains conceptual and educational material only.
It does not include executable code, credentials, or access to real systems.

## Data Handling

- No secrets, tokens, passwords, or private keys are stored in this repository.
- All hostnames, IP addresses, and usernames are placeholders.
- No customer, internal, or proprietary data is included.

## Scope Limitations

- This project does not connect to hosts or networks.
- It does not execute Ansible playbooks or commands.
- It should not be used as a security control or monitoring tool.

## Reporting a Security Issue

If sensitive information is discovered by mistake,
please open an issue or contact the repository owner so it can be removed.
